本地安装部署博客

1、本地安装git

下载地址：[https://git-scm.com/downloads](https://git-scm.com/downloads)。
判断安装：搜索cmd，打开终端，输入

`git --version`

如果有版本表示安装完成

2、本地配置SSH并绑定到GitHub

① 生成SSH：git bash中输入

    `ssh-keygen -t rsa -b 4096 -C “1378718921@qq.com”`

② 获取本地的SSH：git bash中输入

    `cat ~/.ssh/id_rsa.pub`

③绑定到GitHub

    打开GitHub->Settings->SSH and GPG keys->New SSH key
    ->输入Title和Key（步骤②获取的ssh）->完成

3、本地安装node.js

下载地址：[https://nodejs.org/zh-cn/](https://nodejs.org/zh-cn/)。

4、本地安装Hexo

终端输入：

`npm install -g hexo-cli`

5、git clone到本地

选择克隆文件放置的文件夹，右击选择`git bash here`，输入

`git clone -b dev 仓库地址 `

6、安装node依赖

进入clone的仓库目录，输入

`npm install `
